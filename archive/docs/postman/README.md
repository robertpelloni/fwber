# Postman collection (OpenAPI)

This folder contains a ready-to-import OpenAPI 3.0 JSON for FWBer.

- File: `fwber-openapi.json` (generated by L5-Swagger)
- Source: `fwber-backend/storage/api-docs/api-docs.json`

How to import into Postman:
1. Open Postman
2. Click Import → File → select `fwber-openapi.json`
3. Choose "OpenAPI 3.0" if prompted
4. Postman will create a collection and environments from servers

Notes:
- Feature-flagged routes (Recommendations, WebSocket, Mercure, Content Generation, Rate Limiting) are off by default. Enable flags in `fwber-backend/.env` to exercise those endpoints.
- After regenerating docs (`php artisan l5-swagger:generate`), re-copy the JSON:

```powershell
Copy-Item -Path "C:\Users\hyper\workspace\fwber\fwber-backend\storage\api-docs\api-docs.json" -Destination "C:\Users\hyper\workspace\fwber\docs\postman\fwber-openapi.json" -Force
```
