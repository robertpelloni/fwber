# Release v2.1.5 — OkCupid Matching Engine

**Date:** June 5, 2026
**Tag:** v2.1.5

## Summary
The v2.1.5 release introduces a high-signal matching engine based on shared values and mutual satisfaction. This replaces simple interest-based matching with a sophisticated geometric-mean heuristic.

## Features
- **Geometric-Mean Compatibility**: Mutual satisfaction algorithm: `Score = sqrt(SatA * SatB)`.
- **Matching Question Hub**: Dedicated UI at `/settings/matching` for multi-choice value questions.
- **Deep Site Integration**: Compatibility badges on profiles and discovery cards.
- **Cyber-Noir Content**: 15+ high-fidelity value questions seeded for immediate engagement.
- **Staging Automation**: Automated staging deployment workflow via GitHub Actions.
